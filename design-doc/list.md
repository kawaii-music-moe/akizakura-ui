# List

並列の項目を連続して見せるコンポーネント。

## 共通方針

- 縦並びを基本とする。横並びは項目数が少ない場合や、アイコンなど視覚的に小さい単位を並べるときに用いる。
- `<ul>` / `<ol>` を使う場合はマーカーを消し（`list-style: none`）、項目間の余白は `gap` で取る。`margin` を積まない。
- アイテム自体を箱として強く囲うのは避け、必要なら項目間の境界で区切る方針を優先する。アイテム単位を箱として扱いたい場合は [card](card.md) として組む。

## 区切り方

| 方式 | 使いどころ |
|---|---|
| 余白のみ（`gap`） | 軽く並べたいとき。標準は [`--space-sm`](spacing.md)。 |
| 余白 + 下線（`border-bottom`） | 項目を明示的に分けたいとき。最後のアイテムには下線を出さない（`:not(:last-child)`）。 |

下線で区切る場合、線は薄く（不透明度 0.2 前後）して情報の主役を奪わないようにする。線は直書きせず [`--divider`](color.md) トークンを使う（テーマ連動。ダークでは白の淡い線に切り替わる）。

## レスポンシブ

- ナビゲーションのように、レイアウト幅に応じて縦↔横を切り替えてよい。切り替えは [breakpoint](breakpoint.md) の 768px を境界に行う。

## アイテムの中身

- アイテム内のアイコン↔文字の隙間など、親フォントサイズに連動させたい余白は `em` で指定してよい（[spacing](spacing.md) の em / rem 使い分け参照）。
- アイテムがリンクの場合、アンカー自体に `display: flex` を当てて中身（アイコン＋テキスト）をまとめてもよい。アンカー自体のスタイルは [typography](typography.md) のリンク規則に従う。
