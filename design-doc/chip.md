# Chip

選択・フィルタ・入力タグなどに使う、対話できる小さなラベル。[badge](badge.md) の「対話版」と位置づける。

## badge との違い

- [badge](badge.md) は**表示専用**の静的ラベル。chip は**押せる・選べる・消せる**コントロール。
- 見た目の基本（面は [`--surface-tint`](color.md)、角丸は [`--radius`](shape.md) で **pill にしない**）は badge と揃える。

## 方針

- 既定は [`--surface-tint`](color.md) の面。選択（`is-selected` / `aria-pressed="true"`）は [`--accent`](color.md) 塗り。
- **選択は色（塗り）だけで示し、選択時にアイコンを足したりしない。** 状態切り替えでチップの寸法が変わると一覧がガタつくため。チェックマーク等を加えるとアイコン分だけ幅も高さも変わってしまう。
- ホバー・フォーカスで操作可能なことを示す（枠線 accent / [`--border-focus`](color.md) のリング）。
- 先頭にアイコン、末尾に削除（×）を添えられる。アイコンの有無で高さが変わらないよう、チップは `min-height` でアイコン分の高さを常に確保する（テキストのみのチップとアイコン付きチップが同じ高さに揃う）。削除は chip 内の素のアイコンボタンにする。
- 面を使いたくない場面は `chip--outline`（枠線のみ）。

## 使い方の注意

- フィルタ群では「中立（未選択）」を基準にし、選択されたものだけ accent にする。多色を混在させない（[badge](badge.md) と同じ）。
- 単なるジャンル表示など、押せないものは chip ではなく [badge](badge.md) を使う。
