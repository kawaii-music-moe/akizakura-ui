# Color

色は CSS 変数として `:root` に定義し、UI 側からは必ずトークン経由で参照する。

## ベース

| トークン | 値 | 用途 |
|---|---|---|
| `--bg` | `#ffffff` | ページ背景。カード等のサーフェスも基本的にここに載る |
| `--surface-tint` | `rgba(0, 0, 0, 0.03)` | カード等に薄く背景色をつけたいときの塗り |
| `--border` | `#000000` | 既定の境界線 |
| `--border-focus` | `rgba(180, 120, 200, 0.7)` | フォーカス時の境界線 |
| `--divider` | `rgba(0, 0, 0, 0.2)` | 余白だけでは足りないときの薄い区切り線（`--border` より淡い）。リストの下線などに使う |

## テキスト

| トークン | 値 | 用途 |
|---|---|---|
| `--text` | `#1a1a1a` | 本文 |
| `--text-sub` | `#666666` | 補足・キャプション |
| `--text-muted` | `#999999` | プレースホルダ・無効化された要素 |

## アクセント

| トークン | 値 | 用途 |
|---|---|---|
| `--accent` | `#c87ddb` | 主要アクセント（サイトタイトルの `.`、強調、塗り） |
| `--accent-2` | `#e8a0f0` | アクセントの明るい派生 |
| `--accent-dark` | `#9a4fbf` | アクセントの暗い派生 |

## 状態色

| トークン | 値 | 用途 |
|---|---|---|
| `--danger` | `#d9534f` | エラー・警告 |
| `--success` | `#4caf7d` | 成功 |

## リンク

| トークン | 値 | 用途 |
|---|---|---|
| `--link-hover` | `var(--accent-dark)` | アンカーのホバー色 |

## ダークモード

上の表はライト（既定）の値。ダークは同じトークンを `:root` で上書きして切り替える。トークンの**意味**は変えず、**値だけ**を差し替える方針とする（UI 側のコードはどちらのテーマでも変更不要）。値の出典は kawaii-music.moe 本体（portfolio）のダークパレット。

| トークン | ダークの値 |
|---|---|
| `--bg` | `#09080f` |
| `--surface-tint` | `rgba(255, 255, 255, 0.04)` |
| `--border` | `#1f1c2d` |
| `--border-focus` | `rgba(200, 150, 230, 0.8)` |
| `--divider` | `rgba(255, 255, 255, 0.14)` |
| `--text` | `#f2f0f7` |
| `--text-sub` | `#a99fb8` |
| `--text-muted` | `#736a8b` |
| `--accent` | `#d59cea` |
| `--accent-2` | `#ecb4f5` |
| `--accent-dark` | `#c87ddb` |
| `--link-hover` | `var(--accent-2)`（ダークでは明るい派生に向ける） |

`--danger` / `--success` はダークでも視認性が保てるためライトと同値のまま据え置く。

### 切り替えルール

- OS が dark 設定のときは自動でダークにする（`@media (prefers-color-scheme: dark)`）。
- ただし利用側が明示的に `<html data-theme="light">` を付けた場合はライトに固定する。
- 明示トグルは `<html data-theme="dark">` / `data-theme="light"` で OS 設定を上書きできる。
- `data-theme` の付与・永続化（localStorage 等）と初期描画前の反映は**利用側 JS の責務**。CSS 側はその状態に応答するだけで、フレームワーク自体は JS 非依存を保つ。

## 運用ルール

- 色は直接記述せず、必ずトークン経由で参照する。
- 不透明度が必要なトークン（`--border-focus`、`--surface-tint`、`--divider` など）は例外として `rgba()` 記法を使う。それ以外は16進数で統一する。
- 新しい色を導入するときは、まずこのドキュメントにトークンとして追加してから実装する。
- 新しい色トークンを足したら、ライトとダークの両方の値を必ずこのドキュメントに併記する。
