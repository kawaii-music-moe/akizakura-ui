# Color

色は CSS 変数として `:root` に定義し、UI 側からは必ずトークン経由で参照する。

## ベース

| トークン | 値 | 用途 |
|---|---|---|
| `--bg` | `#ffffff` | ページ背景。カード等のサーフェスも基本的にここに載る |
| `--surface-tint` | `rgba(0, 0, 0, 0.03)` | カード等に薄く背景色をつけたいときの塗り |
| `--border` | `#000000` | 既定の境界線 |
| `--border-focus` | `rgba(180, 120, 200, 0.7)` | フォーカス時の境界線 |

## テキスト

| トークン | 値 | 用途 |
|---|---|---|
| `--text` | `#1a1a1a` | 本文 |
| `--text-sub` | `#666666` | 補足・キャプション |
| `--text-muted` | `#999999` | プレースホルダ・無効化された要素 |

## アクセント

| トークン | 値 | 用途 |
|---|---|---|
| `--accent` | `#c87ddb` | 主要アクセント（サイトタイトルの `.`、強調、塗り） |
| `--accent-2` | `#e8a0f0` | アクセントの明るい派生 |
| `--accent-dark` | `#9a4fbf` | アクセントの暗い派生 |

## 状態色

| トークン | 値 | 用途 |
|---|---|---|
| `--danger` | `#d9534f` | エラー・警告 |
| `--success` | `#4caf7d` | 成功 |

## リンク

| トークン | 値 | 用途 |
|---|---|---|
| `--link-hover` | `var(--accent-dark)` | アンカーのホバー色 |

## 運用ルール

- 色は直接記述せず、必ずトークン経由で参照する。
- 不透明度が必要なトークン（`--border-focus`、`--surface-tint` など）は例外として `rgba()` 記法を使う。それ以外は16進数で統一する。
- 新しい色を導入するときは、まずこのドキュメントにトークンとして追加してから実装する。
- ダークモードは現時点で対象外。将来対応する場合はトークン体系を再設計する。
